## 使用方法

1. 先从[项目讲解与面试话术](./02_项目讲解与面试话术.md)学会整体故事。
2. 按上表逐项学习，每项都要能回答“是什么、解决什么、项目怎么用、为什么选、边界是什么”。
3. 最后看[简历项目描述优化记录](./09_简历项目描述优化记录.md)，核对哪些话可以直接写在简历上，哪些只能作为后续演进回答。

判断原则：代码和证据已经支持的能力可以用完成时；只有设计或预留接口的能力必须明确说成目标架构、后续优化或生产化方向。
