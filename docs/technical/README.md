# 技术专题索引

本目录保存路由、动作分析和工具系统的详细设计。它用于回答深入追问，不是第一次学习入口。

第一次学习请先看[项目文档总导航](../README.md)，面试准备请进入[智能体岗位面试学习中心](../interview/README.md)。

## 当前状态与设计方案怎么区分

| 标记 | 含义 |
|---|---|
| 当前实现 | 已经能在代码、接口或测试中找到依据 |
| 当前边界 | 原型已经打通，但能力或质量仍有限 |
| 后续优化 | 设计建议，不代表已经完成 |
| 旧版归档 | 只用于追溯，不参与当前面试口径 |

## 使用建议

- 面试官问总体架构：看[技术深挖与白板](../interview/06_技术深挖与白板.md)。
- 面试官追问某个模块实现：再进入本目录对应专题。
- 需要核对代码现状：以[项目总览](../项目总览.md)为准。
- 需要核对测试与评测：以[项目证据](../项目证据.md)为准。
- 需要查接口字段：以[接口说明](../接口说明.md)为准。

## 旧版资料说明

`interview-archive/` 内的文件已经退出当前学习路线，可能包含过时架构、旧测试数字或早期计划。除非追溯历史决策，否则不应阅读，更不能直接作为当前面试答案。
