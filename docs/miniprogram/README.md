# 微信小程序文档索引

本目录只负责微信小程序端，不代表后端智能体能力本身。

## 阅读顺序

| 顺序 | 文档 | 用途 |
|---:|---|---|
| 1 | [实现状态](./实现状态.md) | 当前已经完成什么、还缺什么 |
| 2 | [设计说明](./设计说明.md) | 页面、组件、状态和接口怎样设计 |
| 3 | [验收计划](./实施计划.md) | 当前剩余验收项和推荐验证顺序；文件名为历史沿用 |

## 当前定位

小程序负责发送对话请求、展示流式输出和执行状态、查询历史，以及上传动作图片或视频。

## 当前边界

- 代码级主链路已完成；“端到端完成”仍需开发者工具和真机验收后才能使用。
- 微信开发者工具和真机完整联调尚未完成。
- 公网部署需要 HTTPS、合法域名和 WebSocket 配置。
- 弱网、断线重连、权限拒绝和不同基础库版本仍需验收。

后端当前状态看[项目总览](../项目总览.md)，接口字段看[接口说明](../接口说明.md)。
